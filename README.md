ENG
SaveMoneyApp is a desktop application written in Python using the PyQt library, designed for personal financial management. It is a tool dedicated to Windows users, enabling effective tracking of financial income and expenses. User data is stored in a local database, ensuring data persistence and the ability to analyze financial records over time.

Main Features of SaveMoneyApp:
Adding Financial Income:
Allows users to record income across various categories (e.g., salary, additional sources of income).
Automatically calculates the total income for each month.
Expense Management:
Users can categorize expenses (e.g., food, housing, entertainment, transportation).
Quick addition of expenses by specifying the amount, category, and optional description.
Monthly Data Organization:
Financial data is organized by months, enabling easy analysis of monthly income and expenses.
Users can reset monthly data when a new month begins, while historical data remains stored in the database.
Database Integration:
Data is stored in a local SQLite database, ensuring persistence and security.
Option to export or create backups of financial records.
User-Friendly Interface:
Designed for simplicity and functionality, making it accessible even to less tech-savvy users.
Clear separation into sections: income, expenses, data analysis, and settings.
Support for Multiple Financial Categories:
Predefined categories for income and expenses.
Ability to add custom categories as per user needs.
Analysis and Reports:
Displays summaries and monthly statistics (e.g., total expenses, monthly balance).
Graphical visualization of data (e.g., bar charts or pie charts) generated with matplotlib.
Installation and Usage:
The application is delivered as an installer for Windows, enabling easy setup on the computer.
System requirements: Windows 10 or newer, Python with the necessary libraries (PyQt5, SQLite).
Technologies Used:
Python: the programming language.
PyQt5: for creating the user interface.
SQLite: as the database.
matplotlib: for generating charts and graphs.
To get started, download the application, install the required tools (as described above), and start managing your finances effortlessly! 😊

PL
SaveMoneyApp to desktopowa aplikacja napisana w Pythonie z wykorzystaniem biblioteki PyQt, stworzona z myślą o zarządzaniu osobistymi finansami. Jest to narzędzie dedykowane użytkownikom systemu Windows, pozwalające na efektywne śledzenie swoich wpływów finansowych i wydatków. Dane użytkownika są przechowywane w lokalnej bazie danych, co zapewnia trwałość zapisów i możliwość analizy w dłuższym okresie czasu.

Główne funkcjonalności aplikacji SaveMoneyApp:
Dodawanie wpływów finansowych:

Możliwość wpisywania przychodów w różnych kategoriach (np. pensja, dodatkowe źródła dochodu).
Automatyczne sumowanie wpływów dla każdego miesiąca.
Zarządzanie wydatkami:

Użytkownik może przyporządkowywać wydatki do różnych kategorii (np. żywność, mieszkanie, rozrywka, transport).
Szybkie dodawanie wydatków z podaniem kwoty, kategorii i ewentualnie opisu.
Podział danych na miesiące:

Dane finansowe są segregowane według miesięcy, co umożliwia łatwą analizę miesięcznych wydatków i wpływów.
Przy rozpoczęciu nowego miesiąca istnieje możliwość zresetowania danych miesięcznych, przy zachowaniu historycznych zapisów w bazie danych.
Integracja z bazą danych:

Dane są zapisywane w lokalnej bazie danych SQLite, co zapewnia trwałość i bezpieczeństwo informacji.
Możliwość eksportu lub tworzenia kopii zapasowej danych.
Intuicyjny interfejs użytkownika:

Aplikacja została zaprojektowana z myślą o prostocie i funkcjonalności, dzięki czemu jest przyjazna nawet dla mniej zaawansowanych użytkowników.
Przejrzysty podział na sekcje: wpływy, wydatki, analiza danych i ustawienia.
Obsługa wielu kategorii finansowych:

Predefiniowane kategorie wydatków i wpływów.
Możliwość dodawania własnych kategorii według potrzeb użytkownika.
Analiza i raporty:

Wyświetlanie podsumowań i statystyk miesięcznych (np. całkowite wydatki, bilans miesięczny).
Graficzna wizualizacja danych (np. wykresy słupkowe lub kołowe).
Instalacja i uruchamianie:
Aplikacja jest dostarczana w postaci instalatora dla systemu Windows, co umożliwia łatwą instalację na komputerze.
Wymagania systemowe: Windows 10 lub nowszy, Python z zainstalowanymi bibliotekami (PyQt5, SQLite).
Technologie użyte w projekcie:
Python jako język programowania z matplotlib do generowania wykresow.
PyQt5 do tworzenia interfejsu użytkownika.
SQLite jako baza danych.
